# 🎵 Spotify Music Analysis Dashboard

## 📌 Project Overview

This project is an interactive **Spotify Music Analysis Dashboard** built using **Power BI**. It analyzes Spotify's Most Streamed Songs 2023 dataset to uncover trends in streaming performance, artist popularity, and music characteristics.

The dashboard enables users to explore streaming statistics through interactive filters and visually appealing charts.

## 🚀 Features

- 🎵 Total Streams KPI
- 🎼 Total Tracks KPI
- 📈 Average Streams KPI
- 📅 Streams by Release Date
- 📆 Tracks & Average Streams by Month
- 📊 Daily Streams Analysis
- ⭐ Artist & Top Streamed Track
- 🏆 Top 3 Most Streamed Tracks
- 🔍 Interactive Filters
  - Track
  - Artist
  - Release Date

---

## 📂 Dataset

**Dataset:** Spotify Most Streamed Songs 2023

The dataset contains information such as:

- Track Name
- Artist(s)
- Release Date
- Streams
- Danceability
- Energy
- Speechiness
- Liveness
- Instrumentalness
- BPM
- Musical Key
- Mode
